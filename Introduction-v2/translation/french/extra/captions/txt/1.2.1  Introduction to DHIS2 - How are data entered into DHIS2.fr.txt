- La saisie des données dans DHIS2 / Introduction à DHIS2 Si nous voulons prendre des décisions réfléchies, basées sur les données qui ont un impact sur les populations nous devons d'abord comprendre comment ces données sont capturées et vérifiées. C'est seulement lorsque nous avons des données fiables que nous pouvons faire confiance à notre analyse pour appuyer la prise de décision. Dans cette vidéo, vous verrez comment les données sont saisies dans le système DHIS2. Dans la vidéo suivante, vous découvrirez comment DHIS2 contribue à assurer la qualité des données. DHIS2 fonctionne comme un entrepôt de données centralisé capable de recueillir des données à partir de nombreuses sources et appareils différents y compris la saisie directe via le Web ou un appareil Android, SMS, et même d'autres sources de données, comme des feuilles de calcul ou d'autres systèmes électroniques. Au sein de la plateforme DHIS2, il existe plusieurs outils de collecte de données que nous pouvons utiliser selon le type de données que nous voulons collecter. Nous pouvons distinguer 3 grandes catégories de données dans DHIS2 également appelées modèles de données, les données numériques ou agrégées, les données individualisées anonymes, et les données individuelles avec identifiants. Passons brièvement en revue chacun de ces modèles. Le premier modèle de données est celui des données numériques ou agrégées. Et nous utilisons l'application d'entrée de données pour capturer les données de ce type. Par exemple, disons que 10 enfants ont reçu leurs vaccins aujourd'hui dans un établissement de santé donné dans le cadre d'un programme de vaccination et on veut saisir cette information dans DHIS2. Quelles sont les étapes que nous devons suivre pour saisir ces données ? Allons sur la plateforme et voyons comment cela fonctionne. Après vous être connecté, allez sur l'icône des applications dans DHIS2 et ouvrez l'application "Saisie de données". Ensuite, utilisez le champ de recherche pour trouver l'établissement. Dans la liste déroulante des ensembles de données, sélectionnez le programme de vaccination. Et enfin, sélectionnez la période que vous souhaitez mettre à jour. Pour saisir les données, cliquez sur la cellule à renseigner et saisissez les valeurs. Notez que les formulaires ainsi que la configuration et la hiérarchie du système de santé sont entièrement personnalisables au sein de la plateforme. Nous y reviendrons plus loin. La deuxième catégorie de données ou modèle de données saisit les données anonymes individualisées également appelées "Événements". Et nous utilisons l'application "Saisie" pour enregistrer les données liées aux événements. Par exemple, disons que nous voulons enregistrer un nouveau cas de COVID-19 dans un établissement de santé donné. Quelles sont les étapes à suivre pour saisir ces données dans DHIS2 ? Revenons à la plateforme. Pour saisir les données d'un événement, sélectionnez l'application "Saisie" à partir de l'icône des applications. Ensuite, recherchez l'établissement, et sélectionnez le programme d'événements. Dans ce cas, les cas de COVID-19. Et ensuite, cliquez sur le bouton "Nouveau". Une fois que vous avez rempli le formulaire, les données sont stockées dans DHIS2. Notez que lors de la saisie des événements, les données saisies sont anonymes, et le suivi des cas individuels n'est pas possible dans le cadre de ce modèle. Le 3ème type de données sont les données individuelles avec des identifiants qui peuvent être suivis dans le temps. Ce modèle est également connu comme le modèle de données de suivi. Nous utilisons l'application "Saisie Tracker" pour enregistrer et saisir ce type de données. En utilisant ce modèle de données, il est possible d'identifier une personne par son nom, son numéro de téléphone, ou une combinaison de tout autre identifiant. Nous utilisons ce modèle lorsque nous voulons faire un suivi des personnes pour suivre leurs prestations et effectuer une gestion des cas au niveau individuel. Par exemple, disons que nous voulons enregistrer un patient avec le COVID-19 pour faciliter la surveillance de la maladie. Quelles étapes devrez-vous suivre pour saisir ces données ? Comme dans les exemples précédents, allez dans l'icône "applications". Et dans ce cas, ouvrez l'application "Saisie Tracker". Ensuite, sélectionnez l'établissement, sélectionnez le programme, et cliquez sur "Enregistrer". Le formulaire contient des données qui identifient les individus et suivent les prestations qu'ils reçoivent. Pour résumer, il existe trois grandes catégories de données ou de modèles de données dans DHIS2, les données numériques ou agrégées, les données individualisées anonymes ou événements, et les données individuelles avec identifiants ou le modèle de données de suivi . Et pour chacun, il existe un outil de saisie spécifique; "Saisie de données" pour les données agrégées, l'application "Saisie" pour les événements, et l'application "Saisie Tracker" pour les données de suivi.
