- L'entrée de données dans DHIS / Introduction à DHIS2 Si nous voulons prendre des décisions éclairées basées sur les données qui ont un impact sur les gens, nous devons d'abord comprendre comment ces données sont capturées et vérifiées. C'est seulement lorsque nous avons des données fiables que nous pouvons faire confiance à notre analyse pour soutenir la prise de décision. Dans cette vidéo, vous verrez comment les données sont saisies dans le système DHIS2. Dans la vidéo suivante, vous découvrirez comment le DHIS2 contribue à assurer la qualité des données. Le DHIS2 fonctionne comme un entrepôt de données centralisé capable de capturer des données à partir d'un nombre de sources et de dispositifs différents y compris la saisie directe via le Web ou un appareil Android, SMS, et même d'autres sources de données, comme des feuilles de calcul ou d'autres systèmes électroniques. Au sein de la plateforme DHIS2, il existe plusieurs outils de collecte de données de collecte de données que nous pouvons utiliser selon le type de données que nous voulons capturer. Nous pouvons distinguer 3 grandes catégories de données dans le DHIS2 également appelées modèles de données, les données numériques ou agrégées, les données individualisées anonymes, et les données individuelles avec identifiants. Passons brièvement en revue chacun de ces modèles. Le premier modèle de données est celui des données numériques ou agrégées. Et nous utilisons l'application d'entrée de données pour capturer les données de ce type. Par exemple, disons que 10 enfants ont reçu leurs vaccins aujourd'hui dans un établissement de santé donné dans le cadre d'un programme d'immunisation, et on veut saisir cette information dans le système DHIS2. Quelles sont les étapes que nous devons suivre pour saisir ces données ? Allons sur la plateforme et voyons comment cela fonctionne. Après vous être connecté, allez sur l'icône des applications dans le DHIS2 et ouvrez l'application "entrée de données". Ensuite, utilisez le champ de recherche pour trouver l'établissement. Dans la liste déroulante des ensembles de données, sélectionnez le programme d'immunisation. Et enfin, sélectionnez la période que vous souhaitez mettre à jour. Pour saisir les données, cliquez sur la cellule "valeur des données" et saisissez les valeurs. Notez que les formulaires ainsi que la configuration et la hiérarchie du système de santé sont entièrement personnalisables au sein de la plate-forme. Nous y reviendrons plus loin. La deuxième catégorie de données ou modèle de données saisit les données anonymes individualisées également appelées événements. Et nous utilisons l'application Capture pour enregistrer les données des événements. Par exemple, disons que nous voulons enregistrer un nouveau cas de COVID-19 dans un établissement de santé donné. Quelles sont les étapes à suivre pour saisir ces données dans le système DHIS2 ? Revenons à la plateforme. Pour saisir les données d'un événement, sélectionnez l'application Capture à partir de l'icône des applications. Ensuite, recherchez l'établissement, et sélectionnez le programme de l'événement. Dans ce cas, les cas de COVID-19. Et ensuite, cliquez sur nouvel événement. Une fois que vous avez rempli le formulaire, les données sont stockées dans DHIS2. Notez que lors de la saisie des événements, les données saisies sont anonymes, et le suivi des cas individuels n'est pas possible dans le cadre de ce modèle. Le 3ème type de données c'est les données individuelles avec des identifiants qui peuvent être suivis dans le temps. Ce modèle est également connu comme le modèle de données de suivi. Nous utilisons l'application "Tracker Capture" pour enregistrer et saisir ce type de données. En utilisant ce modèle de données, il est possible d'identifier une personne par son nom, son numéro de téléphone, ou une combinaison de tout autre identifiant. Nous utilisons ce modèle lorsque nous voulons faire un suivi des personnes pour suivre leurs services et effectuer une gestion des cas au niveau individuel. Par exemple, disons que nous voulons enregistrer un patient avec le COVID-19 pour faciliter la surveillance de la maladie. Quelles étapes devrez-vous suivre pour saisir ces données ? Comme dans les exemples précédents, allez dans l'icône "applications". Et dans ce cas, ouvrez l'application "Tracker Capture". Ensuite, sélectionnez l'établissement, sélectionnez le programme, et cliquez sur enregistrer. Le formulaire contient des données qui identifient les individus et suivent les services qu'ils reçoivent. Pour résumer, il existe trois grandes catégories de données ou de modèles de données dans le DHIS2, les données numériques ou agrégées, les données individualisées anonymes ou événements, et les données individuelles avec identifiants ou le modèle de données tracker. Et pour chacun, il existe un outil de saisie spécifique; l'entrée de données pour les données agrégées, l'application Capture pour les événements, et l'application Capture Tracker pour les données de suivi.
