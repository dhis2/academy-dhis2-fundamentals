Details

  • Data Element Group: Malaria
  • Data Elements: Malaria RDT performed, Malaria RDT Positive
  • Data Dimension: Age (above/below 5 years)
  • Period: Quarterly – Jan 2015 – Dec 2016
  • Organization Unit: Trainingland Facilities

*Brief Note: In this table, there should not be more Malaria RDT positives then Malaria RDT performed. If this is occurring, then there is likely some issues with the quality of the data. For example, in Afghan Dispensary for the period Oct - Dec 2015 in both the < 5 and >= 5 age group, we can see that there is more Malaria RDT positives then Malaria RDT performed. Malaria RDT positives are a direct result of performing an RDT test to detect malaria, and therefore the number of Malaria RDT performed should always be >= the number of Malaria RDT positives.
