Orientation vers les catégories

Dans le DHIS2, les éléments de données, tels que les "cas de paludisme" ou les "visites de soins prénatals", sont souvent divisés en désagrégations telles que l'âge ou le sexe. Cela permet d'effectuer une analyse détaillée et d'améliorer la prise de décision. Par exemple, en utilisant les désagrégations par âge, nous pouvons déterminer le nombre de cas de paludisme confirmés dans des groupes d'âge particuliers. Pour créer ces désagrégations, DHIS2 utilise un concept appelé Catégories. Cette vidéo couvre la conception du modèle de catégories dans DHIS2.
