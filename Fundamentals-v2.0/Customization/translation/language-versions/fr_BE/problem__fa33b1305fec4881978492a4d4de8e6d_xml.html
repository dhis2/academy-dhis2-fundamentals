Solution

Pour créer un ensemble de données avec un formulaire de section, vous devez d'abord créer un formulaire par défaut et configurer les informations de base sur l'ensemble de données, notamment son nom, sa période et ses éléments de données, puis l'affecter aux unités d'organisation. La deuxième étape consiste à créer les sections. Pour chaque section, vous devez définir un nom et affecter des éléments de données.
