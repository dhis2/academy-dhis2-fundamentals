Solution

Lorsque vous modifiez le numérateur ou le dénominateur dans la boîte de construction de formule, si l'expression n'est pas valide (par exemple, s'il manque un élément de données après un opérateur mathématique), vous obtiendrez le message "Expression is not well-formed" et vous ne pourrez pas cliquer sur "Done" pour enregistrer l'expression.
