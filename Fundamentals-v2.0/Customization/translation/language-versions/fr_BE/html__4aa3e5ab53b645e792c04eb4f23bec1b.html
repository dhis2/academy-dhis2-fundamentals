Qu'y a-t-il dans ce module?

In DHIS2, data elements are sometimes further disaggregated into other categories, such as age or sex. In this module, you will learn to use a DHIS2 tool to create these disaggregations and the underlying principles behind the design of this tool.

Objectifs d'apprentissage:

À la fin de ce module, vous devriez être en mesure de:

  • Identifier comment les catégories sont conçues
  • Créez des désagrégations qui peuvent être appliquées à vos éléments de données.

Comment serez-vous évalué?

A la fin du module, vous trouverez 2 activités notées:

  • Quiz: 4 questions à choix multiples, ou vrai ou faux. Vous aurez entre 1 et 3 tentatives pour répondre correctement à chaque question. Vous trouverez le nombre de tentatives autorisées par question à côté du bouton d'envoi. Ce quiz compte pour 3% de votre note finale.
  • Laboratoire: Il vous sera demandé d'effectuer des tâches spécifiques dans notre instance de formation DHIS2. Pour réussir le laboratoire, vous devrez effectuer correctement toutes les étapes. Ce laboratoire contribue à 16% de votre note finale.

Combien de temps durera ce module?

Ce module devrait prendre environ 1 heure et 30 minutes à compléter.
