Aperçu

Dans cette activité, vous allez créer un ensemble de données avec un formulaire de saisie de données par défaut. Vous allez configurer ses informations de base, telles que le nom, la période de collecte, les éléments de données associés à l'ensemble de données et les unités d'organisation où les données seront saisies.

Instructions

Créer un ensemble de données : Formulaire par défaut

  1. Connectez-vous à votre compte dans DHIS2 Customization Practice.
  2. Ouvrez l'application Maintenance
  3. Sélectionnez l'onglet Ensemble de données et choisissez Ensemble de données dans le menu de gauche.
  1. Cliquez sur le signe plus (+) bleu en bas à droite pour créer un nouvel ensemble de données.
  2. Remplissez les champs:
    1. Nom: Vos initiales_RMNCAH
    2. Type de période: Mensuel
    3. Combinaison de catégories: Aucune
  3. Ajouter des éléments de données
    1. Ajoutez les éléments de données:
      • Vos initiales_RMNCAH-ANC 1ère visite
      • RMNCAH-ANC 4ème visite
      • RMNCAH-ANC 8ème ou plus visite
      • RMNCAH-Livraison en établissement
      • RMNCAH- Accouchement par césarienne dans l'établissement
      • RMNCAH-Contraception pour la première fois
    2. Conseil : vous pouvez les trouver en faisant défiler la liste des éléments de données disponibles ou en utilisant le champ de recherche pour les trouver. Déplacez les éléments de données un par un en double-cliquant dessus, ou en cliquant sur l'élément de données et en utilisant la flèche vers la droite pour le déplacer vers la case sélectionnée. Vous pouvez également sélectionner plusieurs éléments de données ou une plage d'éléments de données en utilisant les boutons commande/contrôle ou shift, puis en cliquant sur la flèche vers la droite pour déplacer les éléments vers la case sélectionnée.
  4. Affectez toutes les installations de votre hiérarchie à l'ensemble de données.
    1. Vous pouvez attribuer des unités d'organisation en cochant la case située à côté de l'unité d'organisation.
    2. Ou vous pouvez utiliser le champ "Sélectionner les niveaux", choisir "Installation" et cliquer sur "Sélectionner".
  5. Cliquez sur "Enregistrer".

    Vous avez maintenant créé un ensemble de données avec un formulaire par défaut. Dans l'activité suivante, vous allez créer des titres de formulaire de section et organiser les éléments de données dans les différentes sections.
