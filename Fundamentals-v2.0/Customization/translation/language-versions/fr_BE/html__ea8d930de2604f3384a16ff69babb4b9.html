Partie 1: Créez l'ensemble de données

Instructions

Créez l'ensemble de données sur le paludisme dans le système DHIS2 en utilisant le formulaire de collecte de données comme guide.

Remarque : Lors de la création de l'ensemble de données, utilisez vos initiales_Nom afin d'identifier l'ensemble de données que vous avez créé. Par exemple, si vous voulez créer l'ensemble de données "Réduction de la charge du paludisme" et que votre nom commence par "S", votre deuxième prénom par "N" et votre nom de famille par "D", il sera appelé "SND_MAL - Malaria Burden Reduction". Si vous n'avez pas de deuxième prénom, vous pouvez utiliser les deux premières lettres de votre nom de famille. Cela s'applique également au nom court et au code, s'ils sont utilisés.

Conseil: Pour réussir ce laboratoire, vous devez configurer tous les champs correctement. Vous devrez affecter correctement la période, les unités d'organisation et tous les éléments de données qui appartiennent à l'ensemble de données. Créez un ensemble de données avec des sections, des en-têtes de section et ordonnez correctement les éléments de données dans les sections.
