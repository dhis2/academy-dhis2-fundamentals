Aperçu

Dans cette activité, vous allez configurer un indicateur où vous allez utiliser un opérateur mathématique: Vos initiales_RMNCAH - ANC 1-8 taux d'abandon (%).

Instructions

Créez l'indicateur: Vos initiales_RMNCAH - Taux d'abandon des CPN 1-8 (%)

  1. Connectez-vous à votre compte dans DHIS2 Indicators Practice
  2. Ouvrez l'application Maintenance
  3. Sélectionnez l'onglet indicateur et accédez à la page de gestion des indicateurs.
  4. Pour créer un nouvel indicateur, cliquez sur le signe plus (+) bleu en bas à droite.
  5. Remplissez les champs:
    1. Nom: Vos initiales_RMNCAH - ANC 1-8 taux d'abandon (%)
    2. Nom court: ANC 1-8 taux d'abandon (%)
    3. Description: Numérateur : CPN 1ère visite - CPN 8 visite. Dénominateur : La première visite de soins prénatals sert à mesurer la proportion de femmes enceintes qui n'ont pas effectué la totalité des visites recommandées après leur première visite de soins prénatals. Ces données doivent être examinées chaque année dans un scénario typique.
    4. Type d'indicateur: Pourcentage

    Modifier le numérateur

    1. Ouvrez le dialogue de configuration en cliquant sur "Modifier le numérateur".
    2. Ajouter la description "ANC 1 - ANC 8".
    3. Sur le côté droit, assurez-vous que l'onglet des éléments de données est sélectionné. Dans le champ "Noms de recherche", recherchez l'élément de données "CPN 1ère visite".
    4. Double-cliquez sur l'élément de données pour l'ajouter à la boîte de construction de la formule.
    5. Dans la boîte du générateur de formule, cliquez sur le bouton "-" et ajoutez-le à la formule.
    6. Cliquez sur la zone située après le symbole de soustraction dans la boîte du générateur de formule.
    7. Recherchez l'élément de données "8e visite de CPN" et double-cliquez dessus pour l'ajouter à la formule.
    8. Cliquez sur "Terminé".

    Modifier le dénominateur

    1. Ouvrez le dialogue de configuration en cliquant sur "Editer le dénominateur".
    2. Ajouter la description "Première visite de CPN".
    3. Sur le côté droit, assurez-vous que l'onglet des éléments de données est sélectionné. Dans le champ "Noms de recherche", recherchez l'élément de données "CPN 1ère visite".
    4. Au bas de la boîte de dialogue de configuration, confirmez que la formule est correcte et que le système donne une réponse "valide" à la formule.
  6. Cliquez sur "Terminé".
  7. Cliquez sur "Enregistrer".
