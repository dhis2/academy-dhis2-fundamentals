Instructions

Créer l'indicateur: Vos initiales_RMNCAH - Couverture de la 4ème visite de CPN (%) : Remplir les champs configurables

Note: Comme pour les autres types de métadonnées, lorsque vous créez des indicateurs, utilisez vos initiales dans le nom afin d'identifier l'indicateur que vous avez créé. Toutes les métadonnées du système nécessitent des noms uniques et comme d'autres utilisateurs créent les mêmes indicateurs, l'ajout d'initiales vous permettra de sauvegarder votre indicateur. Par exemple, si vous voulez créer un indicateur "Total des cas de paludisme" et que votre nom commence par 'S', votre deuxième prénom par 'N' et votre nom de famille par 'D', il sera appelé "SND_MAL - Total des cas de paludisme". Si vous n'avez pas de deuxième prénom, vous pouvez utiliser les deux premières lettres de votre nom de famille à la place.

  1. Connectez-vous à votre compte dans DHIS2 Indicators Practice
  2. Ouvrez l'application Maintenance
  3. Sélectionnez l'onglet indicateur et accédez à la page de gestion des indicateurs.
  4. Pour créer un nouvel indicateur, cliquez sur le signe plus (+) bleu en bas à droite.
  5. Remplissez les champs:
    1. Nom: Vos initiales_RMNCAH - Couverture de la 4ème visite de CPN (%)
    2. Nom court: Vos initiales_ANC Couverture de la 4ème visite (%)
    3. Description: Numérateur : 4e visite de CPN. Dénominateur : Femmes enceintes attendues. Calcule le pourcentage de femmes enceintes qui se sont rendues dans un établissement pour leur 4ème visite de soins prénatals par rapport au nombre attendu de femmes enceintes au cours de la période sélectionnée.
    4. Cochez la case "annualisé".
    5. Type d'indicateur: Pourcentage

    Modifier le numérateur

    1. Ouvrez le dialogue de configuration en cliquant sur "Editer le numérateur".
    2. Dans le champ de description, écrivez "ANC 4ème visite".
    3. Sur le côté droit, assurez-vous que l'onglet des éléments de données est sélectionné. Dans le champ "Noms de recherche", recherchez l'élément de données "ANC 4ème visite".
    4. Double-cliquez sur l'élément de données pour l'ajouter à la boîte de construction de la formule.
    5. Au bas de la boîte de dialogue de configuration, confirmez que la formule est correcte et que le système donne une réponse "valide" à la formule.
    1. Cliquez sur "Terminé".

    Modifier le dénominateur

    1. Ouvrez le dialogue de configuration en cliquant sur "Editer le dénominateur".
    1. Dans le champ de description situé dans le coin supérieur gauche, mettez "Grossesses attendues (données de population)".
    2. Sur le côté droit, assurez-vous que l'onglet des éléments de données est sélectionné. Dans le champ "Noms de recherche", recherchez l'élément de données "Population des femmes enceintes attendues".
    3. Double-cliquez sur l'élément de données pour l'ajouter à la boîte de construction de la formule.
    4. Au bas de la boîte de dialogue de configuration, confirmez que la formule est correcte et que le système donne une réponse "valide" à la formule.
    1. Cliquez sur "Terminé".

    7. Cliquez sur "Enregistrer".
