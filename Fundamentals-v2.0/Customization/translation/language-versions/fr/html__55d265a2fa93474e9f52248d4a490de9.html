Éléments de données dans la base de données des pratiques de personnalisation du DHIS2

Si vous allez sur la page de gestion des éléments de données dans l'application Maintenance, vous verrez que certains éléments de données existent déjà dans votre système. Il s'agit d'éléments de données qui appartiennent aux programmes du MRNCA et qui seront utilisés comme références dans les démonstrations et les activités. Ces éléments de données sont là pour votre référence ; vous pouvez consulter leurs noms et leurs propriétés dans la liste de l'application Maintenance, mais vous ne pourrez pas les supprimer ou les modifier.
