Aperçu du laboratoire

Dans cette activité notée, vous allez appliquer les connaissances et les compétences que vous avez acquises tout au long du Module 2. Dans la première partie, il vous sera demandé de créer une partie de la hiérarchie des unités d'organisation de Trainingland. Dans la deuxième partie, vous allez regrouper votre unité d'organisation en groupes d'unités d'organisation privés et publics en fonction de la propriété en utilisant les groupes d'unités d'organisation et le jeu de groupes.

Évaluation

Il s'agit d'un laboratoire noté qui compte pour 16% de votre note finale. Vous avez un nombre illimité de tentatives pour compléter ce travail avec succès. Pour obtenir la note de passage, vous devez effectuer toutes les étapes correctement.

Partie 1: Créer les unités d'organisation dans le DHIS2

Instructions

Créez votre hiérarchie d'unités d'organisation en utilisant le modèle ci-dessous. L'unité d'organisation racine est déjà créée pour vous.

Remarque : lors de la création de la hiérarchie, utilisez vos initiales_Nom afin d'identifier les unités d'organisation que vous avez créées. Par exemple, si vous voulez créer une région animale et que votre nom commence par "S", votre deuxième prénom par "N" et votre nom de famille par "D", elle sera appelée SND_Région animale. Si vous n'avez pas de deuxième prénom, vous pouvez utiliser les deux premières lettres de votre nom de famille. Ceci s'applique également au nom court et au code, s'il est utilisé.

Partie 2: Créez les groupes d'unités d'organisation et les ensembles de groupes.

Pour l'ensemble de groupes d'unités d'organisation de type propriété, vous devez créer les deux groupes suivants : privé et public. À l'aide du modèle ci-dessous, créez les deux groupes et l'ensemble de groupes.

Soumettez votre travail

Une fois que vous avez effectué toutes les tâches de cet exercice, soumettez votre travail en cliquant une seule fois sur le bouton "Soumettre" et attendez jusqu'à une minute pendant que votre travail est évalué. Pendant ce processus, vous verrez "Chargement" sur le bouton.

Consulter votre note

Le système évaluera votre travail automatiquement. Vous obtiendrez une note de réussite ou d'échec pour chaque partie de l'exercice. Si vous avez fait une erreur dans cet exercice, vous recevrez un retour d'information spécifique qui vous aidera à comprendre comment réparer votre erreur et réessayer. N'oubliez pas que vous avez un nombre illimité de tentatives pour terminer ce laboratoire.
