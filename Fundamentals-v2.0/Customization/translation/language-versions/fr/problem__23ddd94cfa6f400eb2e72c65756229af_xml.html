Explication

Le formulaire papier comporte des sections spécifiques avec des noms, il faudrait donc le recréer comme un formulaire de section dans DHIS2. Les formulaires par défaut seraient utilisés si un nom spécifique n'était pas nécessaire, car ces formulaires attribuent le nom de la combinaison de catégories utilisée sur les éléments de données comme en-tête de section.
