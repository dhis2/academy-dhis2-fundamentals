Principes de conception du DHIS2

Dans cette vidéo, vous découvrirez trois importants principes de conception DHIS2 applicables aux systèmes de modèles de données agrégées DHIS2. Nous passerons en revue le processus de configuration DHIS2 et les fonctionnalités connexes, nous aborderons le concept de référentiel de données intégré et nous parlerons de l'importance des indicateurs pour l'analyse.
