Explication

Votre utilisateur peut accéder à l'ensemble de données dans le formulaire de saisie et saisir des données pour les unités d'organisation qui lui sont attribuées, à condition que les paramètres de partage des données des options de catégorie dans l'ensemble de données et l'ensemble de données lui-même soient mis à jour sur "Peut saisir et visualiser".
