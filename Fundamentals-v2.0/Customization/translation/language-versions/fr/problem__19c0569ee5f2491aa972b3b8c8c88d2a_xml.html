Solution:

Si un ensemble de groupes d'éléments de données doit être utilisé comme dimension de données dans des applications analytiques, un élément de données ne doit figurer que dans l'un des groupes d'éléments de données de l'ensemble de groupes d'éléments de données. Ceci afin d'éviter le double comptage de ces valeurs dans les analyses.
