Explication

Vous pouvez avoir un élément de données dans le dénominateur avec des données annuelles, comme des estimations de population. Dans ce cas, vous pouvez sélectionner la case annualisation qui appliquera automatiquement le facteur d'annualisation et multipliera la valeur des autres éléments de données.
