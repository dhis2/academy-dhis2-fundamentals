Création d'indicateurs: Taux d'abandon de la CPN

Dans la vidéo précédente, vous avez appris à créer un indicateur dont le numérateur et le dénominateur contiennent tous deux un élément de données. Dans cette vidéo, vous allez apprendre à créer un indicateur un peu plus complexe où le numérateur contient deux éléments de données. Comme vous le savez déjà, vous pouvez ajouter plus d'un élément de données dans le numérateur ou le dénominateur de l'indicateur en utilisant un opérateur mathématique pour former une expression, comme les additionner ou les soustraire l'un de l'autre. En continuant avec notre ensemble de données RMNCAH, nous allons maintenant créer l'indicateur de taux d'abandon ANC1-8.
