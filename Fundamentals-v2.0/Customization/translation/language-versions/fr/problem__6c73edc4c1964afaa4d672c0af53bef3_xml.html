Solution:

Le nom du formulaire est ce que les utilisateurs verront lorsqu'ils saisiront des données dans une application de saisie de données. Par exemple, si nous utilisons "ANC 1ère visite" comme nom de formulaire, les utilisateurs verront ce nom au lieu du nom "RMNCAH - ANC 1ère visite".
