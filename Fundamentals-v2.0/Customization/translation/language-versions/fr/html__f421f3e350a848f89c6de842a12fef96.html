Activités non notées et notées

Dans ce cours, vous aurez plusieurs occasions de vérifier votre compréhension à l'aide d'activités non notées et d'évaluer vos progrès grâce à des activités notées.

Quel type d'activités non notées trouverez-vous dans ce cours?

Il existe trois types d'activités non notées:

  • Questions de contrôle des connaissances
    • Après chaque vidéo, vous trouverez une variété de questions non notées à choix multiple, vrai ou faux et glisser-déposer. Une fois que vous aurez soumis chaque question, vous pourrez lire l'explication au cas où vous auriez besoin de rafraîchir les concepts appris.
    • Les questions de contrôle des connaissances ont pour but de vous aider à vérifier votre compréhension et à renforcer les concepts clés abordés dans les vidéos. Vous aurez un nombre illimité de tentatives pour répondre correctement à chaque question.
  • Activités
    • Dans la plupart des vidéos, vous verrez des démonstrations de tâches effectuées dans le système DHIS2. Pour que vous ayez la chance d'essayer ces tâches, nous avons créé des activités sous forme de guides étape par étape pour vous montrer comment effectuer les mêmes tâches dans notre base de données de formation. Vous pouvez également aller au-delà de nos instructions et explorer la base de données. Ce sera votre terrain de jeu pour interagir avec DHIS2. Notez que des instructions spécifiques sur la façon d'accéder aux bases de données de formation seront fournies avant de commencer chaque activité.
  • Exercices de pré-labos
    • Des devoirs dans lesquels vous réfléchirez au type de personnalisation que vous allez effectuer dans le DHIS2 et à la manière de concevoir ces configurations avant d'interagir avec la plate-forme. Les exercices de pré-laboratoire sont nécessaires pour réussir les laboratoires notés.

Quel type d'activités notées trouverez-vous dans le cours?

Il existe deux types d'activités notées:

  • Quiz du module
    • 4 - 5 Questions à choix multiple et questions de type vrai ou faux que vous trouverez à la fin de chaque module. Les questions portent sur les concepts clés que vous devez apprendre pour réussir le cours. Vous aurez 1 ou 3 tentatives pour répondre à chaque question.
  • Module Labs
    • Il vous sera demandé de montrer vos compétences pratiques en interagissant avec une base de données DHIS2 spécialement créée pour ce cours afin de tester vos connaissances sur la personnalisation. Vous trouverez un laboratoire à la fin des modules 2 à 6. Les laboratoires doivent être complétés dans l'ordre. Cela signifie qu'avant de commencer le laboratoire du module 3, vous devez d'abord terminer avec succès le laboratoire du module 2.

Vous trouverez des instructions spécifiques avant de commencer toute activité notée et non notée.

Comment interagissez-vous avec les questions?

Dans ce cours, vous trouverez plusieurs types de quiz, mais qu'ils soient notés ou non, chaque question comportera les éléments suivants:

  1. Nombre de réponses : En haut de la question, vous verrez si vous devez choisir une seule réponse ou plusieurs.
    1. Questions à choix multiple avec réponses multiples : Ces questions comportent des cases à cocher carrées et vous pouvez sélectionner plusieurs réponses.
    2. Les questions à choix multiple à réponse unique : Ces questions comportent des cases à cocher circulaires et vous ne pouvez sélectionner qu'une seule réponse.
  2. Bouton "Soumettre" : En cliquant sur le bouton "Soumettre", vous pouvez soumettre votre réponse et utiliser une seule tentative.
  3. Nombre d'essais: Vous aurez de 1 à 3 tentatives par question, selon le type de question. Vous verrez combien de tentatives vous aurez par rapport au nombre de tentatives que vous avez utilisées.
  4. Option de sauvegarde: Lorsque vous souhaitez enregistrer votre réponse ou votre progression et y revenir ultérieurement. Pour soumettre votre réponse, vous devez sélectionner "Soumettre" et non "Enregistrer".
  5. Afficher la réponse: Une fois que vous avez utilisé vos tentatives, le bouton "Afficher la réponse" apparaît.
    1. Il vous indiquera la réponse à la question et vous expliquera pourquoi c'est la bonne réponse. Nous vous recommandons de cocher la case "Afficher la réponse" chaque fois que vous n'êtes pas sûr à 100% de votre réponse à la question, même si vous avez répondu correctement.
