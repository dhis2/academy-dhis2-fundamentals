Aperçu

Dans cette activité, vous allez créer un groupe d'éléments de données, Vos initiales_RMNCAH - Soins prénataux, et un ensemble de groupes d'éléments de données, Vos initiales_RMNCAH

Instructions

Créez un groupe d'éléments de données : Vos initiales_RMNCAH - Soins prénataux

  1. Connectez-vous à votre compte dans DHIS2 Customization Practice.
  2. Dans l'application Maintenance, sélectionnez l'onglet "Éléments de données".
  3. Sélectionnez le groupe d'éléments de données dans le menu de gauche
  4. Cliquez sur le signe plus (+) bleu en bas à droite pour ajouter un nouveau groupe d'éléments de données.
  5. Remplissez les champs:
    1. Nom: Vos initiales_RMNCAH - Soins prénatals
    2. Nom court: Vos initiales_RMNCAH - ANC
  6. Attribuer les trois éléments de données de la section sur les soins prénatals
    1. Ajoutez Vos initiales_RMNCAH - ANC 1ère visite (que vous avez créé dans la dernière activité)

    Et ajoutez les deux autres éléments de données de la section ANC.

    1. RMNCAH - ANC 4ème visite
    2. RMNCAH - 8ème visite de l'ANC

    7. Cliquez sur "Enregistrer".

    Créez un ensemble de groupe: Vos initiales_RMNCAH

    1. Naviguez vers l'onglet de gestion des ensembles de groupes d'éléments de données.
    2. Cliquez sur le signe plus (+) bleu en bas à droite pour ajouter un nouvel ensemble de groupes d'éléments de données.
    3. Remplissez les champs:
      1. Nom: Vos initiales_RMNCAH
      2. Nom court: Vos initiales_RMNCAH

      4. Attribuez des groupes d'éléments de données:

      1. Vos initiales_RMNCAH - Soins prénataux
      2. RMNCAH - Accouchement
      3. RMNCAH - Planning familial

      5. Cliquez sur "Enregistrer".

      Maintenant, tous les éléments de données de l'ensemble de données RMNCAH sont ajoutés dans des groupes basés sur le domaine de soins et tous les groupes sont ajoutés dans un ensemble de groupes RMNCAH qui est défini comme une dimension de données dans les applications analytiques.
