Overview

In this activity, you will edit the default data set you created in previous activity from a default from to a section form.

Instructions

Add in sections to your RMNCAH data set

  1. Sign into your account in DHIS2 Customization Practice
  2. Open the Maintenance application
  3. Select the data set tab and choose data set from the left menu
  4. Find your Your initials_RMNCAH data set in the list of available data sets
  5. Click on the actions menu and select ‘Manage sections’
  1. To add a new section, click on the blue plus (+) sign
  2. Enter the name of the section: Antenatal Care
  3. Assign the data elements to the form:
    1. your initials_ANC 1st visit
    2. ANC 4th visit
    3. ANC 8th or more visit
  4. Arrange their order to match the data collection form. Click on the data element you wish to move and use arrows to move it up and down in the selected box
  1. Click ‘Save’
  2. Repeat this process and create two more sections:
    1. Delivery by Age
    2. Family Planning

    Now you have successfully created a section form.

    Note that at the moment neither you nor any other user would be able to enter data into the data set in the data entry application because the data sets are created automatically with no public access. In the next section, you will learn how to change the sharing settings for users to be able capture data in the data set.
