Explanation

For naming conventions, it is best practice to start with a taxonomy that specifies the program that the data element is used for. Extra words should be cut out to simplify the name, and identifiable information should be used towards the beginning of the name to allow for an easy grouping of the elements in analytic and maintenance applications.
