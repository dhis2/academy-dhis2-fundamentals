Overview

In this activity, you will configure sharing settings for the category options and Your initials_RMNCAH data set so that your own user account is able to capture and view the data, and the Academy administrators are able to view the data.

First, you will update the sharing settings for the category options that you created in Module 3:

  • Your initials_10-14 years
  • Your initials_15-19 years
  • Your initials_20+ years

The “Male” and “Female” category options have already been created for you to have the sharing settings preconfigured correctly.

Instructions

Update sharing settings for category options in the Your initials_RMNCAH data set (Your initials_10-14 years, Your initials_15-19 years, Your initials_20+ years)

  1. Sign in to your account in DHIS2 Customization Practice
  2. Open the Maintenance application
  3. Select the category tab and choose category option from the left menu
  1. Find the first category option (Your initials_10-14 years) in the list of category options
  2. Click on the actions menu open the sharing settings dialogue
  1. In the “Add users or user groups” dialogue, type your name and add it to the “Who has access” list
  1. Click on the pencil icon to configure the sharing settings for your user:
    1. Metadata: Can edit and view
    2. Data: Can capture and view
  1. Let’s assign the correct sharing settings for the Academy admins. Type “Academy admins” in the “Add users and user groups” and add the group to the “Who has access” section
  2. Click on the pencil icon to configure the sharing settings for the group:
    1. Metadata: Can view only
    2. Data: Can view only
  1. Click ‘Save’
  2. Repeat this process to update the sharing settings for other 2 age categories
    1. Your initials_15-19 years
    2. Your initials_20+ years

Update the data set sharing settings

  1. In the Maintenance app, click on the data set tab
  2. Click on data set in the left menu and open the data set management page
  3. Find your Your initials_RMNCAH data set, click on the actions menu, and select ‘Sharing settings’ to open the sharing settings dialogue
  1. Add your user account to the “Who has access” section
  1. Click on the pencil icon next to your name and edit the sharing settings:
    1. Metadata: Can edit and view
    2. Data: Can capture and view
  2. Add the “Academy admins” user group to the “Who has access” section
  3. Click on the pencil icon next to the user group name and edit the sharing settings:
    1. Metadata: Can view only
    2. Data: Can view only
  4. Close the sharing settings dialogue. The sharing settings are automatically saved

Confirm that the sharing settings have been applied correctly by entering some data in the data set

  1. Clear your cache so you can access the latest information from the Data Entry app
    1. From the apps menu, select the Browser Cache Cleaner app
    2. In the app, click the button “Select all” then the button “Clear all selected items”
  1. From the apps menu, navigate to the Data Entry app
  2. In the organisation unit hierarchy on the left, select a facility level organisation unit
  1. In the green dimension box, choose the your initials_RMNCAH data set
  2. Select a monthly period
  3. Enter some data in the cells within the form
  4. If your sharing settings are set up correctly, you should be able to enter data
