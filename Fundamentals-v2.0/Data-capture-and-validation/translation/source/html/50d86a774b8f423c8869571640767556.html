Module Lab

This is a graded assessment lab and it is worth 25% of your final grade.

In order to answer many of these questions, you will have to complete a task in the DHIS2 training instance, Trainingland. Sign in to your Trainingland account using the following link: DHIS2 Validate

Select the "submit" button to submit your answers.

If you want to save your current response without submitting it for grading yet, select the "save" button. This allows you to come back later and resume the attempt.
