Data quality during data entry

DHIS2 has built-in data quality tools to help ensure the quality of your data. You can set minimum and maximum data values to be entered to ensure that data entered are consistent and reliable, and outliers can be checked. You can also run validation checks to find any outliers in your data and ensure accuracy. You can run Reporting Rate Summaries to check the timeliness and completeness of your data.
