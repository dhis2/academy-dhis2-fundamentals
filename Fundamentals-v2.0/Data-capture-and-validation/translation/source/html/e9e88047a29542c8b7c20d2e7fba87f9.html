Module Summary

You have now reached the end of the second module for the Data Capture and Validation course. In this module, you have learned:

  • Data Entry app:
    • The Data Entry app is where you enter aggregate data on the web based DHIS2
      • You can access the Data Entry app by selecting it from the Apps Menu
      • You can then open a data set to enter data into by selecting the three dimensions: Organisation Unit, Data Set and Period
    • Valid data is saved automatically to your DHIS2 server when you enter it into the Data Entry app
      • You can enter data by selecting a data element and then entering a value
      • The data element will turn green when the value is saved
    • If DHIS2 does not detect an internet connection, it will save the data locally on the web browsers cache and will upload when it reconnects
      • DHIS2 will display a message at the top of the screen saying “You are online” when it detects a connection or “DHIS2 does not detect a connection; Values will be saved locally”
      • If DHIS2 has any data stored locally and detects an internet connection, you can upload the data by selecting the Upload button that will appear at the top of the screen
    • In order to view the data in other applications, data sets need to be marked as complete

      • You can complete a data set by selecting the Complete button at the bottom of the screen
      • It’s recommended that you use the Run Validation button to check if your data is valid before completing
    • DHIS2 Capture app on Android devices:
      • The DHIS2 Capture app allows you to enter data at any location where you have an Android device
        • You can download the app from the Google Play Store
        • Once downloaded, open it from your home screen
        • Use your DHIS2 server and login details to access your data sets
        • Open a data set by selecting the data sets icon
        • Select your dimensions: Organisation Unit and Period
      • Valid data is saved automatically to your DHIS2 server when you enter it into the DHIS2 Capture app
        • You can enter data by selecting a data element and then entering a value
      • If the DHIS2 Capture app does not detect an internet connection, it will save the data locally and upload when it reconnects
        • You can also attempt to sync the data by selecting the circular arrows on the data set or by accessing the settings and clicking ‘Sync Data Now’
