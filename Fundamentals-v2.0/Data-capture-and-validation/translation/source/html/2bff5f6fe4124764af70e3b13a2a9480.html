Module Summary

You have now reached the end of the first module for the Data Capture and Validation course. In this module, you have learned about:

  • Data models- This course is focused on Aggregate Data
    • Aggregate Data - This course is focused on Aggregate Data
      • The most common type of data in DHIS2
      • Type of data is aggregated sum of values
    • Event Data- Not featured in this course
      • Captures anonymous data
      • Allows the tracking of particular infections
      • Not for following up on
    • Tracker Data- Not featured in this course
      • Similar to event data
      • Allows the tracking of particular infections
      • Data is not anonymous
      • Able to follow up on data over time
