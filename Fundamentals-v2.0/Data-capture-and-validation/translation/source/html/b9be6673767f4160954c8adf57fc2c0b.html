Overview

In this activity, you will run the Follow-Up Analysis tool in the Data Quality app to check for any data marked for follow-up.

Instructions

Image 1 - A Follow-Up Analysis table with items identified for follow-up

Run Follow-Up Analysis:
  1. Sign in to DHIS2 Trainingland (Validation)
  2. Open the Data Quality app from the Apps Menu
  3. Select the Follow-Up Analysis tool
  4. Select your Follow-Up Analysis Dimensions:
    • Select the COVID-19 Surveillance (Aggregate) Data Set from the dropdown menu
    • Select the Animal Region as the Organisation Unit
    • Select the Period: August 1, 2021 - October 31, 2021
  5. Click ‘Follow-Up’
  6. Analyze the results of the Follow-Up Analysis
    • What data elements have been marked for follow-up?
    • Are there any comments?
    • Are min/max limits included?
