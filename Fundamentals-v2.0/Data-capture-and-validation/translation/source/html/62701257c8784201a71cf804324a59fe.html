Data Quality Review Framework

Ensuring data quality is key in maintaining any form of HMIS and DHIS2 has a number of different tools to help you ensure good data quality standards. As you may recall from the Introduction to DHIS2 course, The World Health Organization (WHO) has a non-digital tool called the Data Quality Review Framework. The framework includes five data quality metrics that can be used to ensure good data quality:

  1. Completeness: Is all the data that needs to be there available?
  2. Timeliness: Has the data been entered in a timely manner?
  3. Consistency: Are the data within a plausible range from previous data collection exercises?
  4. Reliability: Are the data consistent on repeated measurement?
  5. Accuracy: Does the data reflect the actual level of what is being collected?
