Batch Validation Rule Analysis

Now that you have seen how you can validate data from a single data set once it has been entered, how can you validate large amounts of data, such as data from multiple data sets, at once? In this video, you’ll learn how the Batch Validation Rule Analysis allows you to ensure data quality across multiple data sets, periods and organisation units. You’ll start by using the DHIS2 Data Quality app to open up the validation rule analysis function then you’ll run a batch analysis and inspect the results to see which rules have been violated.
