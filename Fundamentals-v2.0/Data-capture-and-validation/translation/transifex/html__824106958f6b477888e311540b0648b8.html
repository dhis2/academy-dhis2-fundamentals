How to access DHIS2 Trainingland

To engage with the course, you’ll need to access our specially made DHIS2 training instances:

  • Trainingland (Entry): https://academy.aggregate.dhis2.org/entry
  • Trainingland (Validation): https://academy.aggregate.dhis2.org/validate

Here are the steps needed to access a Trainingland account and start learning by using DHIS2.

The first instance of DHIS2 you will use in this course is the Trainingland instance called “entry”. This will allow you to practice capturing data in a DHIS2 environment, which will be needed for activities and exercises in Module 2. Below, you can find login information to use for web data entry activities. For activities related to the Android Capture app, you will use different login information that will be provided in Module 2.3.

The second instance of DHIS2 used for this course is the Trainingland instance called "validate". In Module 3, you will use this to validate and perform data quality checks on data already entered by the administrator. Please note that any data you enter in "entry" will not affect any of the data existing in "validate" - so you can safely experiement without affecting the experience of your fellow students.

You will use these Trainingland accounts to complete activities throughout this course, and you’ll find the link to the respective Trainingland instance at the top of each activity and assessment lab page. The accounts will only work for activities and assessment labs in this course, and you’ll need to make a new accounts for the other courses on the DHIS2 Online Academy.

If you have any questions or difficulty in accessing a Trainingland account, don’t hesitate to contact us by email fundamentals-support@dhis2.org.

Feel free to explore the various dashboards, apps and functions in Trainingland.
