Overview

In this exercise, you will enter data in DHIS2 using the Data Entry app, use the run validation button and complete a data set.

Instructions

Image 1 - A data set that has passed validation

Value types, validation rules and data set completion:
  1. Sign in to DHIS2 Trainingland (Entry)
  2. Open the Data Entry app from the Apps Menu
  3. Select your Dimensions:
    • Select a facility level Organisation Unit
    • Select a Data Set from the dropdown menu
    • Select a Period of your choice
  4. Enter an invalid data value such as text, characters or decimals. What kind of error message do you receive? What is the required value type of the data element?
  5. Use the information provided in the error message to correct the data value. Enter some additional valid data values in the form in one or more sections
  6. Select the Run validation button at the bottom of the data set
  7. Are there any validation errors? Are any of the data values too high or too low for the data elements? What operator is shown in the expression? Use the information provided in the validation results to correct the errors then run the validation again
  8. Once your data set has passed validation, select the Complete button to mark the data set complete. A dialog box will appear asking you to confirm the data set is complete, click ‘OK’
  9. In order to see the timestamp, you will need to reload the data set. Take note of your organisation unit, data set and period so you can easily find your data set again. When you are ready, refresh the page and re-enter the dimensions of the data set. Can you find the timestamp? What information is included in the timestamp?

  1. In the Data Entry app, select a facility level organisation unit, the Malaria Burden Reduction data set and a period of your choice
  2. Can you identify the two different values types used in this data set? How do the data entry fields differ? Hint: Look at the stock-out section of the form. What kind of data elements do you see? Can you find the Data Element Information Box?
  3. Explore different data sets, organisation units and periods. Can you find a data set that has been marked complete? What is the timestamp? Would the data set be considered on time?
