Overview

In this exercise, you will learn how to use the DHIS2 Capture app offline.

Instructions

Image 1 - DHIS2 Capture app showing that the device is storing data locally

Entering data offline in the DHIS2 Capture app:
  1. Open the DHIS2 Android Capture app and log in using your DHIS2 Trainingland credentials
  2. Open a data set then navigate to a data entry form in the DHIS2 Capture app
  3. Disconnect your WiFi and turn off your mobile data on your Android device
  4. Enter data values in the form
  5. Save the data and return to the data set home screen
  6. Note the gray circular sync arrows next to the data set you have entered data into, this means that the data is saved locally and has not been uploaded to the server. Click on the sync arrows while your device is still offline. What kind of message do you receive?

Image 2 - The DHIS2 Capture app settings, where you sync the data, as well as see the sync configuration that will automatically sync the data when connected

When connected to the internet, there are three different ways to sync the data in the DHIS2 Capture app with your DHIS2 server. Three sync methods:

  1. Let the app upload the data automatically
    1. Your system administrator can set the app to automatically upload data after a certain time interval or at a specified time
  2. Click the circular sync arrows next to the data set
    1. This will instantly upload the data to the server
  3. Open the home screen, select settings, and then select ‘Sync Data Now’
Sync your data set:
  1. Reconnect to your WiFi
  2. Manually sync your data set using the sync arrows. What kind of message do you receive this time?
  3. Click ‘Send’ to send the data to Trainingland. After the synchronization is complete, do the sync arrows still appear next to the data set?
Explore DHIS2

If you’d like additional practice in DHIS2, try these optional, exploratory steps below:

  1. Navigate to the Android app’s settings
  2. In the settings menu, you will find information about synchronization. When was the app last synced? What is the syncing period? Can you find the Sync Data Now button?
