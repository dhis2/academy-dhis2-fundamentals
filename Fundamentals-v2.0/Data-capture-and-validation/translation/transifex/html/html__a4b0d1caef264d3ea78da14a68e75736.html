Overview

In this exercise, you will enter aggregate data in DHIS2 using the Data Entry app.

Instructions

Image 1 - A loaded data set ready to enter data into

How to load a data set:
  1. Sign in to DHIS2 Trainingland using the following URL: and the credentials you used when creating your account
  2. Open the Data Entry app from the Apps Menu
  3. In the green dimension box, select your Dimensions:
    • Select the Organisation Unit from the Organisation Unit Tree on the right by either using the text search field or expanding the hierarchy. Select the facility level Organisation Unit in Food Region (e.g. Brownie Dispensary or Spaghetti Health Centre)
    • Select the Data Set EPI - Immunisation from the dropdown menu
    • Select a Period of your choice
Enter data and navigate a data set:
  1. Click on the first field BCG doses given, < 1 year and enter “2000” (two thousand) as the data value. After typing the data value in the field, press Tab or Enter on your keyboard
    • What error message occurs? What color is the field?
  2. Since the data value is outside the predetermined range for this data element, the field turned orange. Update the data value to a value inside the predetermined range
  3. Now that a valid data value has been entered, the field will turn green (indicating a valid value was entered and saved) and the next available field will be automatically selected
  4. In the next field, enter a character such as the at sign (@) or an asterisk (*)
    • What error message occurs? What color is the field?
  5. Since the data value entered was invalid, the field turned yellow. Now enter a zero or positive integer as the data value in the field (if the number you entered is outside the predetermined range, simply update the data value to value inside the range). Now that a valid data value has been entered, the field will turn green (indicating a valid value was entered and saved) and the next available field will be automatically selected
  6. Practice entering data and navigating through the data set using various navigation keys: use the Tab button to move to the next data field or Shift + Tab to move backwards. You can also use the up and down arrow keys
  7. Notice you are unable to enter data in the gray fields because data entry for those fields has been disabled and locked
  8. If you are working in a long data entry form or only need to enter data for a particular section, you can quickly navigate to the section using the Filter on Section feature in the green dimension box at the top of the data set. Use this feature to filter for the Adverse Events Following Immunisation section of the form
    • How does the data entry form change?

  9. Return to the Filter on Section feature and select ‘Show all sections’

    • How many sections are in this form?
