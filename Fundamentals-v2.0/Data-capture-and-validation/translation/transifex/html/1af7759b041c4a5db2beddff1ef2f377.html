In this module, you will learn how to enter data into a DHIS2 server. You will start by learning how to open a data set in the DHIS2 Data Entry app, then how to navigate the form, and how to run validation checks on your data. Following this, you will learn how to enter data into a DHIS2 data set using the DHIS2 Capture app for Android mobile devices. After learning how to enter data into both the web app and mobile app, you will learn how to enter data offline into DHIS2.

Learning Objectives

By the end of this module, you should be able to:

  • Enter data into a DHIS2 server using a web browser with the Data Entry app
  • Enter data into a DHIS2 server using an Android device with the DHIS2 Capture app for Android devices
  • Enter data into DHIS2 while offline
  • Run a data validation test on your data
  • Complete a data set

How will you be evaluated?

At the end of the module, you will find 2 graded activities:

  • Quiz: 10 multiple choice, true or false, and drag and drop questions randomly selected from a question bank. You will have 3 attempts to pass the quiz with a score of 80% or higher. This quiz contributes to 20% of your final grade.
  • Assessment lab: you will asked to perform specific tasks in our DHIS2 training instance. After completing these tasks, you will have to answer multiple choice questions to assess your comprehension of the exercise. This assessment lab contributes to 25% of your final grade.

How long will this module take?

The module should take about 1 hour 50 minutes to complete.
