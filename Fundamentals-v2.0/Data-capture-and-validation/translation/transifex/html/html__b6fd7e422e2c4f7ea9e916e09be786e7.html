What is in this module?

What does data capture look like in DHIS2? What sort of data can you collect?

In this module, you’ll be introduced to some of the different concepts covered throughout this course. You’ll look at the different types of data you can enter, as well as a use case of data capture with DHIS2.

Learning objectives

By the end of this module, you should be able to:

  • Distinguish the data models used in DHIS2
  • Relate the DHIS2 data models with their data capture tools

How will you be evaluated?

At the end of the module there is a graded quiz. It contributes 10% to your final grade, and you will have 3 attempts to complete it.

How long will this module take?

The module should take about 30 minutes to complete.
