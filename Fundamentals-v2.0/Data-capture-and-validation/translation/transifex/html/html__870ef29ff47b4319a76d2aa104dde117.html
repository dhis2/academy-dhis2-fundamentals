Overview

In this exercise, you will audit aggregate data in the DHIS2 Data Entry app. You will also identify how to validate and complete the data within the Data Entry app.

Instructions

Image 1 - The Data Element Information Box

Navigating the Data Element Information Box:
  1. Sign in to DHIS2 Trainingland (Entry)
  2. Open the Data Entry app from the Apps Menu
  3. In the green dimension box, select your Dimensions:
    • Select a facility level Organisation Unit, such as Cardinal Hospital Gateway PHC or Banana Health Centre
    • Select the Data Set EPI - Immunisation from the dropdown menu
    • Select a Period of your choice
  4. In the data set, double click on a data value field, such as BCG doses given, < 1 year or IPV doses given, ≥ 1 year, to show the Data Element Information Box:
    • At the bottom of the box, you can view the Data Element History. The chart shows the values entered for this data element and allows you to quickly identify any statistical outliers
    • You can enter a comment for other users to view in the comment box and save it by clicking ‘Save comment’
    • Click the star button to mark the data element for follow-up analysis. This will be discussed later in the course. (Note if you do not see the star button, close the Data Element Information Box and enter a data value into the field then reopen the Data Element Information Box)
    • In the upper right-hand side of the box, the min/max limits for the data element is displayed
    • The Audit Trail tab (next to the Data Element History tab) allows you to see a list of modifications that has been made to the data element, including who has made the changes and when they were made
  5. Press the X icon in the upper right-hand corner to exit out of the Data Element Information Box

Image 2 - Metadata Information Pop-up, Print Buttons, and the Complete, Incomplete and Run Validation Buttons

Metadata Information Pop-up, Print Buttons, and the Complete, Incomplete and Run Validation Buttons:
  1. The Metadata Information Pop-up is displayed while you are in the Data Entry app. It is displayed at the upper right-hand corner of the screen and shows the organisation unit, period, and the specific data element (and the disaggregation if applicable) you are entering data into. Click into different data value fields and watch how the Metadata Information Pop-up updates accordingly
  2. In the upper right-hand corner of the form there are two print buttons. You can print a blank copy of the form or you can print the form as it currently appears
  3. In the bottom left-hand corner, you will see a green bar with three buttons: Complete, Incomplete and Run Validation
    • The Run Validation button will perform a quality check on your data that will show any invalid data
    • The Complete button will mark your data set as complete
    • The Incomplete button reverses the Complete button
Explore DHIS2

If you’d like additional practice in DHIS2, try these optional, exploratory steps below:

  1. Choose another data set, facility level organisation unit and period of your choice
  2. Practice entering some data and using the keyboard functions to navigate through the data set
  3. Look at the history of different data elements. Can you find any outliers?
  4. Can you find any data elements with saved comments or that have been marked for follow-up?
  5. Are there any min/max limits?
  6. Can you find any data sets marked as complete?
