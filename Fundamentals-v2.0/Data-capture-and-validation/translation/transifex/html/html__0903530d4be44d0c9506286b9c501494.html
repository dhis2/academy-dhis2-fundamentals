Overview

In this activity, you will run the Reporting Rate Summary tool in the Reports app to evaluate your data sets for completeness and timeliness.

Instructions

Image 1 - A Reporting Rate Summary table

Run a Reporting Rate Summary Report:
  1. Sign in to DHIS2 Trainingland (Validation)
  2. Open the Reports app from the Apps Menu
  3. Select Reporting Rate Summary
  4. Select Reporting Rate Summary Dimensions:
    • Select the Insect District as the Organisation Unit
    • Select the RMNCAH Data Set from the dropdown menu
    • Select the six-monthly Period of January-June 2021
  5. Click ‘Get Report’
  6. Analyze the results of the Reporting Rate Summary
    • Which organisation units have a Reporting Rate on Time of less than 100%?
    • Which organisation units had the same number of Actual Reports and Expected Reports?
  7. Using the same data set and period dimensions, run the Reporting Rate Summary for the Fish District
    • Which organisations are not completing their data sets in a timely fashion?
  8. >Take note of the results so you can compare the results to the last 6 months of 2021. When you are ready, update the period to July-December 2021
    • How do the results differ?
    • Did any organisation units improve their Reporting Rate on Time?
