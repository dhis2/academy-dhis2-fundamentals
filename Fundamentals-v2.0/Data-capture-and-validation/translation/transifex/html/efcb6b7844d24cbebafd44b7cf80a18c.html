Overview

In this exercise, you will select a data set and then enter data into the data set in the DHIS2 Capture app.

Instructions

Image 1 - The DHIS2 Capture app home screen

Opening a data set:
  1. Open the DHIS2 Android Capture app and log in using your DHIS2 Trainingland credentials
  2. Upon logging in, you will see which data sets you to have access to in DHIS2. These are represented by the colored icons on the home screen
  3. You can tap the small blue i icon in the bottom right corner of the data set icon to see the description of the data set
  4. Open the RMNCAH data set

Image 2 - A data set dimension selection screen in the DHIS2 Capture app

Load the data entry screen:
  1. The data set will show a list of all the previous data that have been entered into the data set by organisation unit and period
  2. Beside their name, you will see a blue pencil or a green checkmark. The blue pencil means the data set is not yet completed and the green checkmark means it is completed
  3. To make a new entry, click the + icon in the bottom right of the screen
  4. Select a facility level organisation unit (e.g. Ostrich Health Centre or Apple District Hospital) and a period of your choice then select ‘Next’ to open the data entry form

Image 3 - The data entry screen with data entered and ready to save

Entering, validating and saving data:
  1. Use the top bar to navigate to the section of the form you want to enter data in
  2. Select a data entry field and then enter some data
    • Try both the Next button and clicking directly into the data value cell to see which method you prefer for data entry
    • Notice as you enter data and move between fields, the system gives you a message saying “Value is saved!” to let you know that the data value has been saved
  3. When you have finished entering data, select the save icon in the bottom right corner
  4. In the dialog box, you will be notified when the save is complete and asked if you want to check the data quality, select ‘Yes’. DHIS2 will inform you if there are any errors in your data that need to be corrected
    • If you need to correct any data, make the changes then select the save icon and run the data quality check again
    • If all validation rules have passed, you get a message saying “Everything looks good!” and asked if you want to complete the data set, select ‘Yes’
Explore DHIS2

If you’d like additional practice in DHIS2, try these optional, exploratory steps below:

  1. Open a data set
  2. Select an organisation unit and period of your choice. For the organisation unit, try using a filter to narrow the results
  3. Explore the various sections of the data entry form
  4. Enter some very large data values in the form. When you save and check the data quality, what kind of error message do you get? Are you able to fix the errors from the information provided to you?
  5. On the home page of your chosen data set, can you find your recent entry? Does it have a blue pencil icon or a green checkmark icon?
  6. Explore the Malaria Burden Reduction data set to find different data element value types. Depending on the data element’s value type, a different data entry tool will appear when you are entering data for that data element. For example, a numeric data value type will bring up a number keypad on your device as the data entry tool. Can you find a data element with the value type other than numeric? How are the data entry tools different?
