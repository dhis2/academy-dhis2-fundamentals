Overview

In this exercise, you will explore how the Data Entry app can save data locally when DHIS2 does not detect an internet connection.

Instructions

Image 1 - The Data Entry app, after reconnecting to the internet, will prompt you to upload your data

Enter data offline in the Data Entry app:
  1. Sign in to DHIS2 Trainingland (Entry)
  2. Open the Data Entry app from the Apps Menu
  3. In the green dimension box, select your Dimensions:
    • Select the facility level Organisation Unit in Animal Region
    • Select the Data Set EPI - Immunisation from the dropdown menu
    • Select a Period of your choice
  4. Note the yellow box at the top of the screen saying “You are online” appears when you open the app
  5. When you are ready, disconnect from the internet but do not refresh or close your web browser
  6. Enter a few data values in the form
    • When the Data Entry app no longer detects an internet connection, a yellow box at the top of the screen will appear saying, “You are offline, data will be stored locally”
    • Notice the data value field still changes to green, indicating the data value has been saved, because DHIS2 will save the data locally on your DHIS2 browser cache
  7. Reconnect your device to the internet
  8. At the top of the screen, the Data Entry app will display another message, “There is data stored locally, please upload to the server” with an Upload button next to the message
  9. Click the Upload button to upload any stored data to DHIS2 Trainingland. At the top of the screen, a yellow box will appear saying, “Upload to server was successful” confirming that the data have been successfully uploaded
