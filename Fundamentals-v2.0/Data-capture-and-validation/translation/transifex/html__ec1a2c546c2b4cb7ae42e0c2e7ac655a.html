Overview

In this activity, you will use the Data Set Report tool in the Reports app to view raw or aggregated data that have been entered into DHIS2 by data set.

Instructions

Image 1 - A loaded Data Set Report

Run a Data Set Report:
  1. Sign in to DHIS2 Trainingland (Validation)
  2. Open the Reports app from the Apps Menu
  3. Select Data Set Report
  4. Select Data Set Report Dimensions:
    • Select the RMNCAH Data Set from the dropdown menu
    • Select the Food Region as the Organisation Unit
    • Select a quarterly Period of your choice
  5. Click ‘Get Report’
  6. Analyze the results of the Data Set Report
    • What kind of data element disaggregations can you identify?
    • Can you determine the total number of ANC 1st visits from this report?
    • How many TT 2 doses were given to pregnant women this quarter?
    • Which age category had the highest amount of deliveries in a facility?
    • Are there data elements with no data values entered?

  7. Download the generated report in PDF and open the PDF. Notice the organisation unit and period is clearly listed above each table

Explore DHIS2

If you’d like additional practice in DHIS2, try these optional, exploratory steps below:

  1. In the Reports app, generate a Data Set Report for Animal Region to compare the results you saw in the previous steps for Food Region. Does the Animal Region have a higher or lower total of ANC 1st visits compared to the Food Region?
  2. Generate a Data Set Report with a data set, organisation unit and period of your choice. Download the generated report in Excel format and analyze the results. Do you prefer viewing the results in DHIS2, PDF or Excel?
