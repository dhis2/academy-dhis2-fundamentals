Feedback

Organisation unit groups and organisation unit group sets allow for classifying organisation units by criteria other than the main administrative hierarchy, such as type of ownership, type of facility etc. In analytics, they are useful for viewing data by groups to see if trends differ (for example, private vs public health facilities).
