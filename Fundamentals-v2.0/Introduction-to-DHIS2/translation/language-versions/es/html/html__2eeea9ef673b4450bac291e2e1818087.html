Qué: Elementos de datos e indicadores

En este video aprenderás sobre la dimensión "Qué" de DHIS2: los elementos de datos. Aprenderás las principales características de los elementos de datos, que definen los tipos de datos capturados en DHIS2. También verás las diferencias entre los elementos de datos y los indicadores,los cuales se calculan utilizando elementos de datos. Encontrarás ejemplos de cada uno que te ayudarán a aclarar las diferencias entre ambos.
