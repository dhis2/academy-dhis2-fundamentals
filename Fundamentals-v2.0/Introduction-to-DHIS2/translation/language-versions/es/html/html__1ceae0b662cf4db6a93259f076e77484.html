DHIS2 alrededor del mundo

Desde que DHIS2 se lanzó en los años 90, ha sido implementado en más de 70 países alrededor del mundo y con nuevos países sumándose cada año. El siguiente mapa interactivo muestra dónde y cuándo se ha implementado DHIS2.

Selecciona los elementos del menú lateral para ver el alcance geográfico de las diferentes implementaciones de DHIS2, tanto como SISTEMA DE GESTIÓN DE INFORMACIÓN DE SALUD (HMIS) para la Vigilancia del COVID-19 y la Vacunación del COVID-19 , para la recolección de datos individuales con el Tracker, para el ingreso de datos con teléfonos móviles con la aplicación DHIS2 Android, para paquetes de datos digitales estandarizados de la OMS, y como Sistema de Información de Gestión Educativa (EMIS). En cada visualización, selecciona los países destacados para conocer más sobre sus implementaciones específicas.
