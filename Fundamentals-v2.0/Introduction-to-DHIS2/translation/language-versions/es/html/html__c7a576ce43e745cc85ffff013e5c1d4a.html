¿Qué contiene este módulo?

¿Qué es DHIS2? ¿Para qué sirve? ¿Cómo se utiliza para ayudar a satisfacer las necesidades del mundo real?

En este módulo, descubrirás qué es DHIS2, qué hace y cómo ayuda a gestionar los datos para fundamentar la toma de decisiones. Comenzarás viendo cómo se capturan, revisan, analizan y comparten los datos en DHIS2. Luego, revisarás algunos casos de uso que muestran cómo se utiliza DHIS2 en diferentes campos, como la salud y la educación

Objetivos de aprendizaje

Al final de este módulo serás capaz de:

  • Reconocer cómo se puede utilizar DHIS2 para capturar, analizar y compartir datos para tomar decisiones informadas.
  • Entender qué es un caso de uso en el contexto de DHIS2.
  • Identificar ejemplos del uso de DHIS2 en programas de salud pública y otros campos.

¿Cómo se te evaluará?

Al final del módulo hay un cuestionario de evaluación. Este cuestionario representa el 40% de tu nota final y tendrás un número ilimitado de intentos para completarlo.

¿Cuánto durará este módulo?

El módulo debería durar unos 60 minutos.
