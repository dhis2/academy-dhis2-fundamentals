Contrôles des connaissances, quiz et laboratoires

Dans cette capture d'écran, vous pouvez voir un aperçu d'une évaluation de type Module Quiz. Lorsque vous entrez dans un Module Quiz, un Labo ou un Contrôle des connaissances de l'unité, vous verrez:

  1. Le type d'évaluation que vous suivez:
    1. Un contrôle des connaissances est un test non noté qui permet de vérifier votre compréhension d'une vidéo ou d'une activité après que vous l'ayez terminée.
    2. Un quiz de module est un test noté à la fin d'un module pour tester vos connaissances sur les points abordés dans le module.
    3. Un module de laboratoire est une activité notée ; vous recevrez les détails d'une instance de DHIS2 à laquelle vous devrez vous connecter, et les questions vous demanderont de réaliser des activités pour trouver la réponse.
  2. You will be informed if the quiz is graded or ungraded, and what percentage of the course certificate the quiz is worth.
  3. Le nombre de tentatives dont vous disposez pour terminer le quiz.
  4. Le nombre de points que vaut la question par rapport au poids total du quiz.
  5. Le champ de la question et celui de la réponse. Il y a 3 types de questions utilisées dans ce quiz:
    1. Les questions à choix multiple à réponse unique : Elles comportent des cases à cocher circulaires et vous ne pouvez en sélectionner qu'une seule.
    2. Questions à choix multiples avec réponses multiples : Elles comportent des cases à cocher carrées et vous pouvez en sélectionner plusieurs.
    3. Les réponses par glisser-déposer : il s'agit souvent de diagrammes avec des images ou du texte dans des cases, et vous devez faire glisser la réponse vers la partie correcte du champ.
  6. Le bouton Soumettre, ainsi que le nombre de tentatives dont vous disposez pour répondre à la question. Certaines questions n'autorisent qu'une seule tentative, d'autres en autorisent trois.
