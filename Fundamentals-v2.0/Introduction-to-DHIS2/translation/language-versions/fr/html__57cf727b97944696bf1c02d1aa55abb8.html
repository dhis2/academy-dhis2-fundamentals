Quiz du module 3

Il s'agit d'un quiz noté qui vaut 30% de votre note finale.

Vous avez un nombre illimité de tentatives pour répondre à chaque question.

Notez que pour réussir le cours, vous devez obtenir une note de passage de 80%.

Pour des informations plus détaillées sur la navigation et la notation des quiz, reportez-vous à l'unité Évaluer vos progrès : Contrôles des connaissances et quiz dans la section 0.3 Utilisation de la plate-forme Online academy.

Cliquez sur le bouton "soumettre" pour envoyer vos réponses.
