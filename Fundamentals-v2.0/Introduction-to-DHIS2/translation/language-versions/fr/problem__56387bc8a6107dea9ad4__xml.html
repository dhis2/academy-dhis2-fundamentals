Commentaires

Un entrepôt de données est correct. Le DHIS2 fonctionne comme un entrepôt de données centralisé capable de capturer des données à partir d'un certain nombre de sources et de dispositifs différents, y compris la saisie directe via le web ou un dispositif androïde, les SMS et même d'autres sources de données telles que les feuilles de calcul ou d'autres systèmes électroniques.
