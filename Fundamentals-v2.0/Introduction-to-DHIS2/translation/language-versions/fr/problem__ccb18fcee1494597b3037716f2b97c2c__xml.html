(Sélectionnez 1 option) A. Port de données B. Entrepôt de données C. L'entraînement des données D. Autoroute des données

Solution

L'entrepôt de données est correct. Le DHIS2 fonctionne comme un entrepôt de données centralisé capable de capturer des données à partir d'un certain nombre de sources et de dispositifs différents, y compris la saisie directe via le web ou un dispositif Android, les SMS, et même d'autres sources de données telles que les feuilles de calcul ou d'autres systèmes électroniques.
