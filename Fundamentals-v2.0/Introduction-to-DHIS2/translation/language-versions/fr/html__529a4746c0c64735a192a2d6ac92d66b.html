Quiz du module 1

Il s'agit d'un quiz noté qui vaut 40% de votre note finale.

Vous avez un nombre illimité de tentatives pour réussir ce quiz.

Cliquez sur le bouton "soumettre" pour envoyer vos réponses.

Si vous souhaitez vérifier votre note finale pour ce quiz, veuillez vous reporter à l'onglet "Progression" en haut de la page. Vous pourrez y voir votre résultat pour chaque quiz et votre progression globale. Notez que pour réussir le cours, vous devez obtenir une note de passage de 80 %.
