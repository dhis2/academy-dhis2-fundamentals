Quoi : Éléments de données et indicateurs

Dans cette vidéo, vous allez découvrir la dimension "Quoi" du système DHIS2 : les éléments de données. Vous découvrirez les principales caractéristiques des éléments de données, qui définissent le type de données à saisir. Vous apprendrez également la distinction entre les éléments de données et les indicateurs, qui sont calculés à partir des éléments de données. En cours de route, vous trouverez des exemples de chacun d'entre eux pour clarifier la distinction entre les deux.

Concepts clés

  • Élément de données
  • Indicateur
