Commentaires

Pour créer une visualisation dans le DHIS2, vous devez ajouter trois dimensions : l'élément de données ou le "quoi", la période ou le "quand" et l'unité d'organisation ou le "où". Les personnes et le formulaire de données ne sont pas des dimensions pour créer des visualisations de données dans le DHIS2.
