Solution

Dans le DHIS2, tous les indicateurs doivent avoir un numérateur, un dénominateur et un facteur multiplicateur. Cependant, si le calculateur est un nombre total, par exemple, "Total des visites ANC" (ANC1 + ANC2 + ANC3 + ANC4), alors le dénominateur et le facteur multiplicateur seront tous deux égaux à 1.
