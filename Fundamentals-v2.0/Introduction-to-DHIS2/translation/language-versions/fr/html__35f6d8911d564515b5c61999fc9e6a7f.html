DHIS2 et HISP

Comme vous l'avez vu dans les leçons précédentes, le DHIS2 a été lancé à l'origine comme un projet du Health Information Systems Programme, ou HISP, en Afrique du Sud en 1994. Mais que fait le HISP maintenant ? Et quel est son rapport avec le DHIS2?

Dans cette vidéo, vous découvrirez comment le HISP travaille en tant que mouvement mondial pour renforcer les systèmes d'information sur la santé. Vous apprendrez également comment le HISP est impliqué dans le développement de DHIS2, et aide à mettre en œuvre DHIS2 dans le monde entier. Cliquez sur le bouton de lecture ci-dessous pour commencer.
