Qu'y a-t-il dans ce module?

Qu'est-ce que le DHIS2 ? Que fait-il ? Comment est-il utilisé pour aider à répondre aux besoins du monde réel?

Dans ce module, vous découvrirez ce qu'est le système DHIS2, ce qu'il fait et comment il aide à gérer les données pour éclairer la prise de décision. Vous commencerez par voir comment les données sont capturées, examinées, analysées et partagées dans le système DHIS2. Ensuite, vous examinerez quelques cas d'utilisation qui montrent comment le système DHIS2 est utilisé dans différents domaines, tels que la santé et l'éducation.

Objectifs d'apprentissage

À la fin de ce module, vous serez en mesure de:

  • Reconnaître comment le système DHIS2 peut être utilisé pour saisir, analyser et partager des données afin de prendre des décisions éclairées.
  • Comprendre ce qu'est un cas d'utilisation dans le contexte de DHIS2.
  • Identifier des exemples d'utilisation de DHIS2 dans des programmes de santé publique et d'autres domaines.

Comment serez-vous évalué?

À la fin du module, il y a un quiz noté. Il contribue à hauteur de 40% à votre note finale et vous aurez un nombre illimité de tentatives pour le compléter.

Combien de temps prendra ce module?

Le module devrait prendre environ 60 minutes à compléter.
