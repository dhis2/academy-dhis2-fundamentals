You can make dashboards public or private but you are not able to share a dashboard with specific users or user groups D. Tous les utilisateurs d'un même pays voient les mêmes tableaux de bord. E. Vous pouvez interpréter les résultats du tableau de bord et en discuter avec d'autres utilisateurs.

Solution

Dashboard is typically the first thing users see when they log into DHIS2 so it’s especially useful if users need quick access to information. Typically, a dashboard contains a collection of outputs regarding a specific topic or a program. The data that are shown in a user’s dashboard depends on their own level of access. Interpretations can be made through the dashboard.
