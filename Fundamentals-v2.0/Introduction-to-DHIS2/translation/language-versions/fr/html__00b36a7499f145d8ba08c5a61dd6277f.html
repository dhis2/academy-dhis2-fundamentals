Quand: Périodes

Dans la dernière vidéo de la série sur les blocs de construction du système DHIS2, vous allez découvrir la dimension "Quand" du système DHIS2 : les périodes. Dans le système DHIS2, les périodes font référence au cadre temporel associé aux données. Vous apprendrez les similitudes et les différences entre l'utilisation des périodes dans la collecte des données, où les données sont capturées pendant une période de temps prédéfinie, et l'analyse des données, où les données peuvent être visualisées dans des périodes fixes ou relatives pour produire différents résultats d'analyse.

Concepts clés

  • Period
  • Période relative
  • Période fixe
