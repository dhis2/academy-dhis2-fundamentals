Naviguer sur la plate-forme

Si vous avez besoin d'un rappel sur la façon de naviguer dans l'Académie en ligne DHIS2, veuillez regarder la vidéo suivante pour revoir les bases de la plateforme.

Vous pouvez également télécharger et consulter les directives complètes de navigation sur la plate-forme pour obtenir des informations plus détaillées.

La page du contenu du cours

Dans cette capture d'écran, vous pouvez voir un aperçu des options de navigation les plus utilisées de la plateforme. Lorsque vous sélectionnez la page du contenu du cours, vous voyez:

  1. Le menu de gauche vous montre tous les modules ainsi que leurs sous-sections respectives.
  2. Flèches ou pages qui vous permettent de naviguer dans une sous-section. Elles se trouvent en haut et en bas de la page.
  3. Le fil d'Ariane, pour voir dans quelle unité, sous-section et module vous vous trouvez actuellement.
  4. Dans les onglets supérieurs, vous trouverez des liens pour vérifier votre progression dans les activités notées du cours (Progression), des liens pour lire les questions les plus fréquemment posées sur le cours (FAQ), et des liens pour lire ou télécharger le Guide du participant.
  5. Le bouton "Signets" vous permet d'afficher tout le contenu de vos signets.
  6. Un champ de recherche qui vous permet de rechercher le contenu du cours. Saisissez les termes de la recherche et appuyez sur "Entrée" ou cliquez sur la loupe pour effectuer la recherche.
