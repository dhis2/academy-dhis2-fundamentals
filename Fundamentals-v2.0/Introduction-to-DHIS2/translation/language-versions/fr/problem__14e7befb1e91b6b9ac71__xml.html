(Sélectionnez 3 options) A. Pays B. Programme d'immunisation C. Clinique de santé D. District E. Formulaire de saisie des données sur les hospitalisations et les sorties

Solution

Les pays représentent généralement un niveau dans la hiérarchie des unités d'organisation. Le pays est généralement subdivisé en unités administratives plus petites, telles que des districts ou des régions. Généralement, le niveau le plus bas dans la hiérarchie des unités d'organisation est le niveau de collecte des données où les services sont fournis, comme les cliniques de santé.
