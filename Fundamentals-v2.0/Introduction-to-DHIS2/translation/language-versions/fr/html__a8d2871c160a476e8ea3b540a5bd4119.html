Fondamentaux du DHIS2

Nous souhaitons vous présenter brièvement les cours suivants de la série DHIS2 Fundamentals:

  • Saisie et validation des données dans DHIS2 : Dans ce cours, vous apprendrez à utiliser les applications DHIS2 pour saisir des données agrégées et vérifier leur qualité.
  • DHIS2 Analyse des données : Dans ce cours, vous apprendrez à créer des tableaux croisés dynamiques et à visualiser des données agrégées au moyen de graphiques et de cartes afin de faciliter l'interprétation des données et la prise de décision.
  • Personnalisation du système DHIS2: Dans ce cours, vous apprendrez à connaître les éléments clés qui peuvent être personnalisés dans le système DHIS2 en fonction de votre programme et de votre structure organisationnelle.
  • Fondamentaux des événements DHIS2: Dans ce cours, vous apprendrez à travailler avec des données individuelles en mettant l'accent sur les programmes d'événements, y compris la création de sorties de données d'événements et la configuration des métadonnées des programmes d'événements dans DHIS2.

Comment s'inscrire à ces cours?

Directement sur l'académie en ligne DHIS2. Sélectionnez chaque cours dans le catalogue des cours pour en savoir plus, y compris les éventuels prérequis pour chaque cours spécifique. Vous trouverez un bouton d'inscription sur la page de description du cours pour chaque cours actif.

Quand et dans quel ordre dois-je suivre ces cours?

Il n'y a pas d'ordre spécifique pour suivre les cours, cela dépend de vos propres besoins d'apprentissage. Il n'y a pas non plus de dates spécifiques pour commencer ou terminer, car tous les cours sont autogérés.

Dois-je suivre tous les cours pour être éligible aux académies de niveau 1 ou de niveau 2?

Il n'est pas nécessaire que vous suiviez tous les cours de la série Fondamentaux pour vous qualifier pour les académies de niveau 1 ou de niveau 2. Dans le diagramme interactif suivant, vous verrez l'ordre des cours que nous vous recommandons de suivre en fonction de votre profil/rôle d'utilisateur et de vos intérêts. En sélectionnant un cours particulier, vous en verrez la description, y compris les conditions préalables à ce cours. Pour plus d'informations sur le programme de l'Académie DHIS2, visitez le site Web de DHIS2.
