(Sélectionnez 3 options) A. Unité d'organisation (le "Où") B. Data Set (the “What”) C. Élément de données (le "quoi") D. Période (le "Quand") E. Patients et clients (le "Qui")

Solution

Comme souligné dans la présentation des éléments de base, le système DHIS2 comporte trois dimensions essentielles. L'unité d'organisation fait référence à l'emplacement géographique ou à l'aspect "où", l'élément de données fait référence à "ce" que nous collectons, et la période fait référence à "quand" nous collectons cette information.
