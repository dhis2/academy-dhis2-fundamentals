Questions fréquemment posées (FAQs)

  • Combien de temps faudra-t-il pour terminer le cours ?

    • Nous estimons que le cours durera environ 3 heures. Cela comprend le visionnage de toutes les vidéos, l’exécution des activités connexes et la réalisation de tous les exercices notés. Étant donné qu'il s’agit d’un cours où chacun suit son propre rythme, veuillez noter que cela peut varier légèrement pour chaque apprenant.

  • Y a-t-il un délai pour terminer le cours ?

    • Non. Mais au cas où une date de fin est prévue, vous en serez informé avec suffisamment de temps pour terminer votre travail dans le cours.

  • Comment serai-je évaluée ?

    • Le cours comprend 3 quiz notés que vous trouverez à la fin des modules 1, 2 et 3. Dans ces quiz, vous trouverez 10 questions aléatoires tirées d'une banque de questions et vous aurez un nombre illimité de tentatives pour réussir les quiz.

      Pour réussir le cours, une note globale de 80 % est requise.

  • Quand et où puis-je obtenir mon certificat ?

    • Pour obtenir le certificat du cours, vous devez obtenir une note de passage de 80 % ou plus. Pour vérifier votre note actuelle, allez à la page Progression dans le menu supérieur et vérifiez votre score. Si vous avez déjà obtenu la note de passage, félicitations ! Vous pouvez télécharger votre certificat de cours en allant sur votre tableau de bord et en sélectionnant Mes cours. Vous trouverez la liste des cours auxquels vous êtes inscrit, localisez l'Introduction au DHIS2 et cliquez sur le certificat de cours pour le télécharger ou l'imprimer.

  • C'est la première fois que je prends un cours sur l'Académie DHIS2 en ligne, comment puis-je naviguer sur la plateforme d'apprentissage ?

    • Nous avons préparé un document dans lequel vous trouverez des directives de navigation qui vous aideront à apprendre comment vous déplacer entre les modules, les sous-sections et les unités. Vous trouverez également dans le module 0 un tutoriel vidéo de 5 minutes expliquant comment naviguer sur la plate-forme.

  • Ai-je besoin de me connecter au cours à des moments précis ?

    • Non, tous les contenus et activités sont asynchrones, ce qui signifie que les interactions en temps réel ne sont pas nécessaires pour terminer le cours et que tout le contenu est accessible au moment qui convient le mieux à votre emploi du temps.

  • Que dois-je faire si je rencontre des problèmes techniques ?

    • Pour une assistance technique, vous pouvez contacter les modérateurs du cours pour obtenir de l’aide par e-mail à fundamentals-support@dhis2.org. Pour une réponse plus rapide, n’oubliez pas d’indiquer votre nom d’utilisateur ou l’adresse e-mail que vous avez utilisée pour vous inscrire sur la plate-forme.
