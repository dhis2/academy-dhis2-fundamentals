Explanation

While all visualizations have download options in DHIS2, the list of available download options depends on the visualization type. Pivot tables have table layout download options such as Excel, CSV and HTML, while charts have graphics download options such as Image and PDF. Both pivot tables and charts have Excel download options as well as other plain data source download options.
