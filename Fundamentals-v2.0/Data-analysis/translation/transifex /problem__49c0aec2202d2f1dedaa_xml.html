In the Data Visualizer app, select pivot table as the visualization type. Click on any disaggregation to open the selection box.

>Select all that apply. Add to SeriesAdd to TableAdd to ColumnsAdd to RowsAdd to FilterAdd to Category

Explanation

Within the selection box, you must use the button/down arrow to add the disaggregation to either the columns, rows or filter in the table. Selecting the down arrow next to ‘Add to Columns’ generates the additional options for adding the disaggregation to the rows or filter.
