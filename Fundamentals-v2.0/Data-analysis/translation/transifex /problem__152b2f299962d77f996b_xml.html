In the Data Visualizer app, open the organisation unit dimension. In the organisation unit selection box, you can “Select a level” to display data in the pivot table at a particular level.

Select all that apply. Province Region District Facility Urban

Explanation

In Trainingland, Province and Urban are not organisation unit levels. Trainingland has four levels: Country, Region, District and Facility. These four levels can be found under the “Select a level” dropdown menu in the organisation unit selection box.
