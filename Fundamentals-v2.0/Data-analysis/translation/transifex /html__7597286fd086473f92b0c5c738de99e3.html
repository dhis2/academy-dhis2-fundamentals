Orientation to pivot tables: Organisation unit dimension and disaggregations

The three main dimensions you need to select to create a pivot table are: 1) the data items , the “What” 2) the period, the “When” and 3) the organisation unit, the “Where.” In the previous video, you saw how to edit the What and When dimensions. In this video, you will see how to modify the Where dimension, the organisation unit. You will also see how to add disaggregations to your output.
