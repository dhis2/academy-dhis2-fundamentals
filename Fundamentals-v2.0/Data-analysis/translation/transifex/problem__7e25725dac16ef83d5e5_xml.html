In the Data Visualizer app, select stacked bar as the visualization type. Click on any disaggregation to open the selection box.

Select all that apply. Add to Chart Layout Add to Series Add to Category Add to Rows Add to Columns Add to Filter

Explanation

Within the selection box, you must use the button/down arrow to add the disaggregation to either the series, category or filter in the chart. Selecting the down arrow next to ‘Add to Series’ generates the additional options for adding the disaggregation to the category or filter.
