Explanation

In DHIS2, a map legend conveys the meaning of symbols and colors on a map, helping you to understand and interpret the data on a map. Data from maps can be shown in a table below the map visualization. When a filter is applied in a data table, the filter applies to the data displayed in both the table and the map visualization. In the Style tab of the thematic layer, there are only two options for color legends: automatic and predefined.
